TI BASIC Simulator — Test Programs
===================================
Copy-paste each section into the serial terminal and run with RUN.
Remember to type NEW before each new program.


=== 1. READ / DATA / RESTORE ===

NEW
10 FOR I=1 TO 5
20 READ A,B$
30 PRINT A;B$
40 NEXT I
50 DATA 1,"ONE",2,"TWO",3,"THREE"
60 DATA 4,"FOUR",5,"FIVE"
70 RESTORE
80 READ X,Y$
90 PRINT "FIRST AGAIN:";X;Y$
100 END
RUN

Expected:
 1 ONE
 2 TWO
 3 THREE
 4 FOUR
 5 FIVE
FIRST AGAIN: 1 ONE


=== 2. RANDOMIZE and RND ===

NEW
10 RANDOMIZE
20 FOR I=1 TO 5
30 PRINT INT(RND*100)
40 NEXT I
50 END
RUN

Expected: 5 random integers 0-99 (different each run)


=== 3. ON GOTO ===

NEW

500 NEXT I
510 END
RUN

ONE
TWO
THREE
FOUR

=== 5. CALL KEY ===

NEW
10 PRINT "PRESS ANY KEY (Q TO QUIT)"
20 CALL KEY(0,K,S)
30 IF S<1 THEN 20
40 IF K=81 THEN 100
50 IF K=113 THEN 100
60 PRINT "YOU PRESSED:";K;CHR$(K)
70 GOTO 20
100 PRINT "BYE"
110 END
RUN
